# Mendoza Tour and Transfer - Estructura

## Estructura
- css/base: variables y reset.
- css/sections: estilos por módulo (header, carousel, cards, traslados, turismo-aventura, excursiones, contacto, reservas placeholder, index general).
- css/responsive: ajustes tablet y mobile.
- js: script principal (script.js).
- img: activos agrupados por sección (carrusel, tour, excursiones, etc.), más logo y whatsapp.png.
- *.html: páginas individuales (inicio, servicios, tours, traslados, turismo-aventura, excursiones, contacto).

## Notas
- Todas las páginas referencian la misma hoja de estilos y el mismo JS desde las nuevas rutas organizadas.
- El archivo css/sections/reservas.css es un placeholder para mantener la ruta declarada en los HTML.
- Si agregas nuevos estilos, ubícalos en css/sections o css/base según corresponda.
